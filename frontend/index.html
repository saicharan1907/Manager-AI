Simple File Structures

To get deep AI insights, Manager AI processes standard CSV or Excel files. Simply ensure your files contain these basic columns to unlock the full potential of your dashboard.

inventory_2 Inventory File

  • sku (Unique ID)
  • name (Product name)
  • category (Group tag)
  • quantity (Current stock)
  • price (Unit price)

trending_up Sales File

  • date (YYYY-MM-DD)
  • sku (Product sold)
  • quantity (Units sold)
  • total (Total revenue)
